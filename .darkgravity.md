## Project Description

_agentic_plane_ is a fork of Plane, an open-source project management tool. It includes the Agentic Architecture templates to act as a governance framework for AI-augmented developers.
The platform consists of a Web frontend (Next.js/React) and an API backend (Django/Python) powered by PostgreSQL and Redis.

## Conventions

- Language: Python 3.x, TypeScript 5.x
- Framework: Django (Backend), Next.js (Frontend)
- Test runner: pytest (Backend), vitest/jest/playwright (Frontend)
- Style: PEP 8 / black (Backend), prettier + eslint (Frontend)

## Key Files

- `apps/` or `space/` or `web/` — Frontend source code
- `apiserver/` — Backend API source code
- `CODEX/` — Governance, standards, and blueprints for agentic coding.
- `.agent/` — Context and workflow configurations for agentic behavior.
